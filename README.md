🍅 Temporizador Pomodoro

Um aplicativo simples de temporizador baseado na técnica Pomodoro, desenvolvido em Python com interface gráfica usando Tkinter.

📌 Sobre o projeto

Este projeto foi criado com o objetivo de ajudar na produtividade utilizando a técnica Pomodoro, que alterna períodos de foco com pausas curtas e longas.

A aplicação permite:

- Iniciar ciclos de trabalho
- Fazer pausas automáticas
- Resetar o cronômetro
- Acompanhar o progresso com marcações visuais ✔

🧠 Como funciona a técnica Pomodoro?

A técnica segue este padrão:

1. 25 minutos de foco (trabalho)
2. 5 minutos de pausa curta
3. A cada 4 ciclos → pausa longa (20 minutos)

🚀 Funcionalidades

- ⏱️ Contagem regressiva automática
- 🔄 Alternância entre trabalho e pausas
- 🧮 Controle de ciclos (repetições)
- ✔️ Indicador visual de sessões concluídas
- 🖥️ Interface gráfica simples e intuitiva

🛠️ Tecnologias utilizadas

- Python
- Tkinter (interface gráfica)
